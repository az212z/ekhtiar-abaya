# الأختيار المُبهِر للعبايات — موقع تعريفي

موقع تسويقي عربي (RTL) فاخر لمحل **الأختيار المُبهِر للعبايات** في بريدة.

## المعلومات
- **التصنيف:** محل عبايات فاخرة وتفصيل حسب المقاس
- **المدينة:** بريدة — حي الصفراء، شارع المشاغل
- **التقييم:** 4.7 على خرائط قوقل (100 مراجعة)
- **التواصل:** هاتف 0562727808 · واتساب 966562727808

## سبب الاختيار
محل بتقييم عالٍ (4.7) وحضور قوي وصور حقيقية مميزة لمنتجاته (عبايات وأطقم تخرّج)، بلا موقع إلكتروني — فرصة مثالية لإطلالة رقمية احترافية تعكس فخامة العلامة.

## فكرة التصميم
هوية **"دار عبايات مينيمال معاصرة"**: رمادي فحمي (#2B2A2E) + شمبانيا/بيج (#CBB89C) + عاجي (#F5F1EB) مع لمسات تاوب. تصميم نظيف معماري الطابع يميّزه عن النمط الكلاسيكي الذهبي الحالم. خط عرض ثقيل (Tajawal) مع جسم نصّي نظيف (IBM Plex Sans Arabic)، مساحات بيضاء سخية، وحركات هادئة مصنوعة يدويًا.

## التقنيات
HTML5 + CSS3 + JavaScript (vanilla) — بدون أي مكتبات أو خطوة بناء. يُنشر مباشرة على GitHub Pages.

## الصور
الصور حقيقية للنشاط، مصدرها خرائط قوقل (ملف المشروع). تمت **تنقية** الصور: استُبعدت صور تحمل لافتات نشاطات أخرى (ek-13709 شارع بمحلات أخرى، ek-9234 واجهة "روﭪ ABAYA")، واستُخدمت أربع صور: واجهة المحل وثلاث صور منتجات.

| الملف | الاستخدام |
|------|-----------|
| `ek-23315.jpg` | البطل (Hero) + المعرض — عباية بيج على مانيكان (تحمل علامة المحل) |
| `ek-16447.jpg` | قسم "لماذا نحن" + المعرض — عباية تخرّج عنّابية |
| `ek-6701.jpg`  | المعرض — أطقم تخرّج بالتطريز (تحمل شعار المحل) |
| `ek-10769.jpg` | المعرض — واجهة المحل بلوحته |

## حالة الـ backend
**غير مربوط.** نموذج الطلب تجريبي: يجهّز رسالة واتساب مُعبّأة عبر `wa.me` + يحفظ نسخة في `localStorage` + يعرض إشعار نجاح (toast). لا يوجد خادم يخزّن البيانات.

## التشغيل محليًا
```bash
cd ekhtiar-abaya
python3 -m http.server 8000
# افتح http://localhost:8000
```
